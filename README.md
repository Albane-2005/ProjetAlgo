# Projet Algorithmique 2A - ProjetAlgo

## Description
L'algorithme prépare une boîte (variable) pour stocker du texte, qu'on appelle projetAlgo. Il met la phrase "Bienvenue sur le Projet AlgoBox collaboratif !" dans cette boîte projetAlgo. Il affiche le contenu de la boîte projetAlgo à l'écran. Il demande ensuite le nom et l'âge de l'utilisateur.


## Exécution
1. Ouvrez AlgoBox.
2. Allez dans Fichier > Ouvrir, et sélectionnez : `src/démarrage_projet.algo.alg`.
3. Lancez l'exécution.
